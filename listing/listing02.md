
Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Вывод: 2 1.

В функции test выходным значением является х, а defer вызывается в конце функции,
следовательно когда мы инкрименируем в лямбда функции x, то инкрименируем
мы выходное значение

В функции anotherTest выходным значением является любое значение типа int
(то, которое мы передадим через return) и на момент выполнения defer'а
возвращаемое значение и переменная x – разные вещи

defer откладывает вызов указанной функции до завершения текущей функции, в которой она вызвана.
Вызовы defer складываются в стэк, по этому они вызываются в обратном порядке(Last In First Out) 
```
